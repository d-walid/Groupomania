# Installation du frontend

## Tech Stack

**Server:** React, Redux, Axios, Bootstrap,

## Installation

```bash
cd client
npm install
npm start
```

## Environment Variables

Pour que ce projet fonctionne, ajoutez cette ligne dans un fichier .env Ã  la racine du front.

```bash
REACT_APP_API_URL=http://localhost:5000
```
